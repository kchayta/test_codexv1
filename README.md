# EduNova Classroom

หน้าเว็บตัวอย่างสำหรับคลาสรูมยุคใหม่ที่ผสมผสานเอไอ เกมมิฟิเคชัน และคอมมูนิตี้ เพื่อสร้างประสบการณ์การเรียนที่สนุกและมีส่วนร่วมมากขึ้น

## โครงสร้างโปรเจกต์
- `index.html` – โครงสร้างหน้าเว็บหลัก
- `styles.css` – งานออกแบบและการตกแต่ง
- `script.js` – ฟังก์ชันเชิงโต้ตอบและเครื่องมือเอไอจำลอง

## วิธีใช้งาน
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ที่คุณชื่นชอบ หรือใช้คำสั่งต่อไปนี้เพื่อรันเว็บเซิร์ฟเวอร์แบบง่าย ๆ:

```bash
python -m http.server 8000
```

จากนั้นเปิดเบราว์เซอร์แล้วเข้าไปที่ [http://localhost:8000](http://localhost:8000)
